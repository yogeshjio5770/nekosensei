# NekoSensei — Why We Beat Duolingo for Japanese

## The NekoSensei Learning Loop

Every lesson follows **4 steps** — not just tap-to-translate:

| Step | What you do | Why it works |
|------|-------------|--------------|
| **Learn** | Read with romaji + meaning | Understand first |
| **Listen** | Native TTS pronunciation | Train your ear |
| **Speak** | Mic + speech recognition | Build speaking muscle |
| **Quiz** | Test with animations + haptics | Lock it in memory |

## Fast Learning Features

- **Daily Boost (5 min)** — 5 essential phrases with listen + speak
- **Quick Review (SRS)** — spaced repetition for weak words
- **Conversation Role-Play** — speak real dialogue lines
- **AI NekoSensei Tutor** — ask grammar, get corrections

## vs Duolingo

| Feature | Duolingo | NekoSensei |
|---------|----------|------------|
| Japanese TTS on every word | Limited | ✅ Everywhere |
| Speaking practice | Premium / limited | ✅ Built-in |
| Romaji for beginners | Hidden early | ✅ Always shown |
| 4-step lesson flow | Mostly quizzes | ✅ Learn→Listen→Speak→Quiz |
| JLPT certificates | No | ✅ N5/N4/N3 |
| AI tutor | Duolingo Max ($) | ✅ Groq included |
| Daily 5-min boost | Streak only | ✅ Focused phrases |
